bRando.js

A CSS background property changer capable of transitioning randomly or sequentially. ⌨️ by yak.

Documentation + Code: https://github.com/isaacyakl/brandojs

Distributed under the MIT License.

Code Examples

Background randomizers are created using the bRando.new() method and are attached to select HTML elements by passing a DOM selector string argument. Try it for yourself:

bRando.new('html') bRando.new('div.example') bRando.new('p.example')

Additionally, randomizers may be controlled individually and together. For example, the following methods control all randomizers present on a page:

bRando.nextAll() bRando.pauseAll() bRando.playAll() bRando.removeAll()
