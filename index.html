bRando.js

An HTML element background randomizer. Written with vanilla JavaScript by yak.

Documentation + Code: https://github.com/isaacyakl/bRando

Distributed under the MIT License.

Code

Click a .new() button to create an image rotator in the demo section below:

bRando.new('div.example') bRando.new('p.example') bRando.new('button.example') bRando.new('input') bRando.new('textarea')

Move image rotators to the next image and also pause, play, or remove them all:

bRando.nextAll() bRando.pauseAll() bRando.playAll() bRando.removeAll()
