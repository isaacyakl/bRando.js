jBir

A vanilla javascript Background image rotator library by yak.

Documentation + Code: https://github.com/isaacyakl/jBir

Distributed under the MIT License.

Code

Click a .new() button to create an image rotator in the demo section below:

jBir.new('div.example') jBir.new('p.example') jBir.new('button.example') jBir.new('input') jBir.new('textarea')

Move image rotators to the next image and also pause, play, or remove them all:

jBir.nextAll() jBir.pauseAll() jBir.playAll() jBir.removeAll()
